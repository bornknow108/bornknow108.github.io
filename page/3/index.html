(Zend Framework)headLink とか headScript の初期化

$this->view->headLink() とか
$this->view->headScript() とか使って
css の設定や javascript の設定をしていた場合に
場合によっては、クリアして 再設定したい場合があるとおもいます。
そんなときは、
$this->view->headScript()->exchangeArray(array());
で初期化出来ます。

(headScript は適宜おきかえてください。)
各設定の情報が ArrayObject で管理されているので
ArrayObject のメソッドを利用して、空の配列をセットすることで初期化をします。

dotCloudのCLIをインストール

こちらで紹介されていたdotCloudのCLIインストールで、ちょっと手間取ったのでメモっておく。

ひとまずサインアップして、Installing the CLI してよと言われたので、手順通りにすすめたが、API KEYを入力した後
「error: Authorization rejected」
と言われたまま、なにもできなくなった。

ちょこっと調べたところ、手順のページのAPI KEYの最後のドットが不要だったらしい。

修正方法は、Macの場合だと「~/.dotcloud/dotcloud.conf」内に記載されているapikeyの項目を修正すればOKです。

dotcloud.confのapikeyを修正後、dotcloudコマンドを実行すればちゃんと認証が通ります。
