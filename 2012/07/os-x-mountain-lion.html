OS X Mountain Lion（マウンテンライオン）にアップデートしてみた


OS X Mountain Lion（マウンテンライオン）が購入できるようになっていたので、’早速、会社のPCにダウンロード＆インストールしてみました。’

###まずは、システム設定をのぞいてみる

いくつかアイコンが増えていました。


スクリーンセーバーです。表示の仕方が変更されているような気がします。


通知機能の設定ですね。


音声入力の設定ですね。


システムの音声は「ビクトリア」にしました。


まだまだ、いろんな変更がありそうです。

###とりあえず、Mac Portsの更新から
インストールが完了したので、Mac Portsのselfupdateを実行。
早速怒られました。
ひとまず、XCodeを確認したら、バージョンが古いため起動できませんでした。
しかたがないので、App Storeより最新版をダウンロード＆インストール。
あわせて、Command Line Toolも更新。
Command Line Toolのインストールでも四苦八苦したのですが
こちらの記事を参考にして解決しました。
